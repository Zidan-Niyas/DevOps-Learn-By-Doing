# Contributing Guide

We welcome new DevOps learning resources!

## How to Contribute

1. **Fork this repo**
2. **Add your resource** to the correct category in `README.md`
3. **Open a Pull Request** with a clear title and description

### Resource submission format

- Use the following format: `[Resource Name](URL) - Short Description (if necessary)`

### Issues

- To suggest a new category or raise a problem, please [open an issue](https://github.com/dth99/devops-learn-by-doing/issues/new).

---

Thank you for contributing!
